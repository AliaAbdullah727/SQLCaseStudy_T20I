# 🏏 T20 International Cricket Data Exploration using SQL

## Project Overview

This project explores **T20 International Cricket match data** using SQL to answer business-style analytical questions. The objective was to transform raw match records into meaningful insights by analyzing team performance, winning trends, venue statistics, and match outcomes.

The project demonstrates the use of advanced SQL techniques to perform data exploration and generate actionable insights.

This project was completed as part of the **DevAlpha Data Analytics Internship Program**.

---

# 🎯 Objectives

- Analyze team performance throughout the tournament.
- Rank teams based on wins.
- Evaluate winning margins.
- Investigate head-to-head records.
- Analyze venue performance.
- Calculate team win percentages.
- Explore monthly match trends.
- Practice advanced SQL querying techniques.

---

# 🛠️ Technologies Used

- Microsoft SQL Server
- SQL (T-SQL)

---

# 📂 Dataset

The dataset contains T20 International cricket matches played in **2024**, including:

- Match Date
- Team 1
- Team 2
- Winner
- Winning Margin
- Ground
- Match Result

---

# 📊 Business Questions Answered

The project answers several analytical questions, including:

### 🏆 Team Performance

- Which team won the most matches?
- Which teams have the highest win percentages?
- How many matches did each team play?

---

### 📈 Team Rankings

- Rank teams based on total victories.
- Identify the most successful chasing team.
- Compare head-to-head records between two teams.

---

### ⚡ Winning Margin Analysis

- Team with the highest average winning margin (Runs).
- Team with the highest average winning margin (Wickets).
- Matches with above-average winning margins.

---

### 🏟️ Ground Analysis

- Most successful team at each ground.
- Ground-wise winning distribution.

---

### 📅 Time Analysis

- Month with the highest number of T20I matches.
- Match distribution throughout the year.

# 🔍 Key Insights

Some notable findings from the analysis include:

- Teams were ranked according to total wins using window functions.
- Winning margins were analyzed separately for victories by runs and wickets.
- Certain teams consistently performed better when chasing targets.
- Monthly match analysis revealed periods with the highest tournament activity.
- Ground-specific analysis identified the most successful team at each venue.
- Team win percentages provided a more meaningful performance metric than total wins alone.
